How to read this

For a fair speed comparison, filter to one test case and one OLS selection. “Typical” means the median, so one unusually slow run does not dominate. “Last 5 vs previous 5” compares two five-run windows.

Older runtime rows did not record the test-case file, runway count, layout, or exact parameters. They remain visible for airport and OLS exploration, but are marked rough. New rows store an exact setup code and are marked exact.
